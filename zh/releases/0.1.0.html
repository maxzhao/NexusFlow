NexusFlow 0.1.0 MVP ​

0.1.0 是 NexusFlow 的初始 MVP 版本，确立了桌面应用、UE 插件和 Rust Bridge 协同工作的基础能力。

主要内容 ​

  • NexusFlow 桌面应用基础体验。
  • 与 Unreal Engine 插件的连接能力。
  • AI 对话侧边栏与悬浮球入口。
  • 蓝图读取、解释、生成和修改的初始能力。
  • UE 快速菜单入口。
  • 多模型配置和基础设置中心。

后续变化 ​

0.2.0 在此基础上增加了 MCP Server、外部 Agent 接入说明、Widget Skill MVP，并显著提升蓝图操作稳定性。
