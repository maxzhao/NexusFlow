设置说明 ​

NexusFlow 的设置页面采用 iOS 风格的导航设计，集中管理 AI 模型、LLM 代理、Agent、快捷键、外观等配置。点击侧边栏顶部的齿轮图标即可进入设置。

AI 模型 ​

管理 AI 模型是使用 NexusFlow 的第一步。在这里你可以添加、验证和管理多个 AI 模型。

模型列表 ​

已添加的模型以卡片形式展示，每张卡片显示：

  • 模型名称和 Provider 标签
  • 验证状态：是否通过连接测试
  • 默认标记：标注当前使用的默认模型

点击卡片可展开查看详细信息，包括模型标识、API Key（直接填写时掩码显示；环境变量引用时显示变量名）、Base URL、Temperature 和 Max Tokens。

模型操作 ​

对每个已添加的模型，你可以：

  • 验证模型：测试 API Key 和连接是否正常
  • 设为默认：将该模型设为 AI 对话使用的默认模型
  • 删除模型：移除不再需要的模型配置

添加新模型 ​

点击「添加模型」按钮进入配置表单：

1. 选择 Provider

支持以下 AI 服务提供商：

Provider说明
OpenAIGPT 系列模型
AnthropicClaude 系列模型
Google GeminiGemini 系列模型
DeepSeekDeepSeek 系列模型
智谱 GLM智谱 GLM 系列模型
MiniMaxMiniMax 系列模型
通义千问通义千问 (Qwen) 系列模型
KimiKimi 系列模型
Custom自定义 API 地址

2. 填写基本信息

  • 模型名称（必填）：自定义名称，方便识别
  • API Key（必填）：服务提供商的 API 密钥；也可以填写 ${OPENAI_API_KEY} 这类环境变量引用，真实密钥由后端运行时读取，应用配置中只保存变量名引用
  • 模型标识（必填）：模型的具体型号，如 gpt-5.4、claude-opus-4-6

3. 可选配置

  • Temperature：控制回复的随机性，滑块范围 0 - 2，值越低回复越确定
  • Max Tokens：AI 单次回复的最大 Token 数

Anthropic 协议注意

使用 Anthropic Messages API 协议时，Max Tokens 为必填项。

4. Custom Provider 额外配置

选择 Custom 时，需要额外填写：

  • API 地址：自定义 API 端点 URL
  • 通讯协议：选择与目标 API 兼容的协议
    • OpenAI Compatible
    • OpenAI Responses API
    • Anthropic Messages API
    • Google Gemini API

LLM 代理 ​

LLM 代理是独立的全局设置，位于设置页的「LLM 代理」入口。它只影响 NexusFlow 桌面主程序发往 LLM 服务的请求，不影响 UE 插件、Rust Bridge、MCP Server、窗口通信等内部连接。

可配置项：

  • 启用代理：开启或关闭全局 LLM 代理。
  • 协议：支持 HTTP、HTTPS、SOCKS4、SOCKS5。
  • 主机 / 端口：代理服务器地址，例如 127.0.0.1:10808。
  • 用户名 / 密码：可选认证信息；密码使用 NexusFlow 的加密密钥存储保存，不写入普通设置文件。
  • 测试代理连通性：只测试代理服务器地址和端口是否可连接，不验证目标 LLM 服务是否可访问。

代理规则提示

如果你的本地代理软件启用了「绕过大陆 IP」或类似分流规则，即使 NexusFlow 已经把 LLM 请求发送到本地代理，代理软件仍可能按目标服务器地理位置选择直连。需要强制走代理时，请在代理软件中关闭相应绕过规则或为目标 LLM 服务配置强制代理规则。

Agent 配置 ​

配置 AI 助手的行为参数。

最大迭代次数 ​

设置 AI 在单次对话中的最大工具调用轮数，范围 1 - 1000。

  • 默认值适合大多数场景
  • 复杂的蓝图操作可能需要更多迭代次数
  • 设置过低可能导致 AI 无法完成复杂任务

回复语言 ​

控制 AI 回复使用的语言，独立于界面显示语言：

  • 跟随界面语言：使用与界面相同的语言
  • 简体中文
  • 繁體中文
  • English

TIP

你可以将界面设为英文，但让 AI 用中文回复，反之亦然。

快捷键 ​

自定义 NexusFlow 的快捷键绑定。

全局快捷键 ​

在任何应用中都可触发的快捷键：

  • 切换侧边栏：自定义组合键，用于快速显示/隐藏 NexusFlow 侧边栏

点击快捷键输入框后，直接按下想要的组合键即可录入。

应用快捷键 ​

仅在 NexusFlow 窗口中生效的快捷键：

  • 发送消息：选择 Enter 或 Ctrl+Enter 作为发送消息的快捷键
    • 选择 Enter：按回车直接发送，Shift+Enter 换行
    • 选择 Ctrl+Enter：按 Ctrl+Enter 发送，回车换行

恢复默认 ​

点击「恢复默认」按钮可将所有快捷键重置为默认值。

外观 ​

个性化 NexusFlow 的视觉外观。

主题 ​

三种主题模式：

  • 深色（默认）：VS Code Dark 风格，护眼暗色主题
  • 浅色：明亮的浅色主题
  • 跟随系统：自动跟随 Windows 系统主题设置

语言 ​

界面显示语言：

  • 跟随系统：自动跟随操作系统语言
  • 简体中文
  • 繁體中文
  • English

字体 ​

点击字体选择区域打开全屏字体选择器：

  • 列出系统已安装的所有字体
  • 搜索框快速过滤
  • 每个字体使用自身效果预览
  • 点击即可选用

字号 ​

选择界面文字大小，提供 5 档选择：

  • 12px / 14px / 16px（默认）/ 18px / 20px

TIP

外观设置修改后实时预览，自动保存，无需手动确认。

关于 ​

查看应用信息和管理相关设置。

应用信息 ​

  • 版本号：当前 NexusFlow 版本
  • Tauri 版本：底层框架版本

插件状态 ​

自动检测系统中的 Unreal Engine 安装，并检查 NexusFlow 插件是否已正确安装。

  • 显示检测到的 UE 版本
  • 显示插件安装状态
  • 点击「重新检测」刷新状态

匿名分析 ​

可选的匿名使用数据收集，帮助改善产品。你可以随时开启或关闭此选项。
