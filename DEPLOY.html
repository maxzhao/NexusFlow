NexusFlow 文档站点部署指南 ​

文档站点基于 VitePress 构建，部署到 Cloudflare Pages。

前置条件 ​

  • Node.js >= 18
  • npm
  • Cloudflare 账号
  • Wrangler CLI (可选，用于命令行部署)

本地开发 ​

bash
# 安装依赖
cd docs
npm install

# 启动开发服务器 (支持热更新)
npm run docs:dev
# 或使用 just 命令 (项目根目录)
just docs-dev

# 本地预览生产构建
npm run docs:build
npm run docs:preview
# 或
just docs-build
just docs-preview

构建配置 ​

VitePress 配置 ​

  • 配置文件: docs/.vitepress/config.mts
  • 构建输出: docs/.vitepress/dist/
  • 多语言: 中文 (/zh/) + 英文 (/en/)
  • 主题: 暗色主题

构建命令 ​

命令说明
npm run docs:dev开发服务器
npm run docs:build生产构建
npm run docs:preview预览构建结果

Cloudflare Pages 部署 ​

方式一: GitHub 集成 (推荐) ​

  1. 登录 Cloudflare Dashboard

    • 打开 Cloudflare Pages
    • 点击 Create a project > Connect to Git

  2. 连接 GitHub 仓库

    • 选择 NexusFlow 仓库
    • 授权 Cloudflare 访问

  3. 配置构建设置

    配置项值
    Production branchmaster
    Build commandcd docs && npm install && npm run docs:build
    Build output directorydocs/.vitepress/dist
    Root directory/ (项目根目录)
    Node.js version18 (或更高)

  4. 环境变量 (可选)

    变量值说明
    NODE_VERSION18指定 Node 版本

  5. 点击 Save and Deploy

    • 首次部署会自动触发
    • 后续每次 push 到 master 都会自动部署

方式二: Wrangler CLI 部署 ​

适用于手动部署或 CI/CD 环境。

bash
# 1. 安装 Wrangler
npm install -g wrangler

# 2. 登录 Cloudflare
wrangler login

# 3. 构建文档
cd docs
npm run docs:build

# 4. 部署到 Cloudflare Pages
wrangler pages deploy .vitepress/dist --project-name=nexusflow-docs

# 或使用 just 命令 (项目根目录)
just docs-deploy

方式三: 直接上传 ​

  1. 打开 Cloudflare Pages Dashboard
  2. 点击 Create a project > Direct Upload
  3. 项目名称: nexusflow-docs
  4. 上传 docs/.vitepress/dist/ 目录下的所有文件
  5. 点击 Deploy site

自定义域名 (可选) ​

  1. 在 Cloudflare Pages 项目设置中，点击 Custom domains
  2. 点击 Set up a custom domain
  3. 输入域名 (如 docs.nexusflow.dev)
  4. 按提示添加 DNS 记录 (CNAME)
  5. 等待 DNS 生效和 SSL 证书签发

部署验证 ​

部署完成后，检查以下内容：

  • [ ] 首页可正常访问
  • [ ] 中英文切换正常 (/zh/ 和 /en/)
  • [ ] 搜索功能正常 (VitePress 本地搜索)
  • [ ] 暗色主题正确显示
  • [ ] 所有文档页面链接有效
  • [ ] 静态资源 (图片/Logo) 加载正常

故障排除 ​

构建失败 ​

bash
# 清理缓存后重新构建
rm -rf docs/.vitepress/cache docs/.vitepress/dist docs/node_modules
cd docs && npm install && npm run docs:build

404 错误 ​

  • 检查 build output directory 是否设为 docs/.vitepress/dist
  • 确保 cleanUrls: true 配置与 Cloudflare Pages 兼容

多语言路由问题 ​

  • VitePress 的 locales 配置中 link 路径必须与文件目录结构一致
  • 中文: /zh/ 对应 docs/zh/ 目录
  • 英文: /en/ 对应 docs/en/ 目录

CI/CD 集成 (GitHub Actions) ​

如需通过 GitHub Actions 部署 (替代 Cloudflare Git 集成)：

yaml
# .github/workflows/docs-deploy.yml
name: Deploy Docs

on:
  push:
    branches: [master]
    paths: ['docs/**']

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: 18

      - name: Install & Build
        run: |
          cd docs
          npm install
          npm run docs:build

      - name: Deploy to Cloudflare Pages
        uses: cloudflare/wrangler-action@v3
        with:
          apiToken: ${{ secrets.CLOUDFLARE_API_TOKEN }}
          command: pages deploy docs/.vitepress/dist --project-name=nexusflow-docs

需要配置的 GitHub Secret:

  • CLOUDFLARE_API_TOKEN: 在 Cloudflare API Tokens 创建，需要 Cloudflare Pages:Edit 权限
